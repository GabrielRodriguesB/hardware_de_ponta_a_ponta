<h1>Sobre o projeto:</h1>


* Site criado apenas para fins didáticos e de portfólio, onde mostro o meu conhecimento na linguagem de marcação HTML

* O projeto possui apenas um arquivo HTML (index.html) e o foco desde o início foi de montar a página apenas utilizando o HTML, sem utilizar o JavaScript , CSS (nenhum arquivo de normalize ou reset CSS) ou qualquer outra biblioteca/framework

* Desafio pessoal, pois eu tentei ao máximo saber até onde é possível construir uma página apenas com HTML

* Possui responsividade total (mobile até desktop)

* Feito o teste de compatibilidade no Google Chrome e Mozilla Firefox

* Utilizado a tag **&lt;center&gt;** com o foco de conseguir centralizar o conteúdo sem utilizar o text-align, pois não queria usar o CSS. Essa tag foi descontinuada na versão 4 do HTML, juntamente com a tag **&lt;marquee&gt;**

* Explorei ao máximo os atributos globais e individuais de cada tag (assim como as entidades), semântica, indentação e organização dos containers

* Todos os conteúdos de terceiros (vídeos e áudios) estão com os devidos créditos. Demais imagens estão disponíveis de forma gratuita na internet e livre de direitos autorais
